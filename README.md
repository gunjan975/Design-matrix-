# UTS Prediction Model

This notebook trains a linear regression model (with/without hidden layers) to predict UTS (MPa) using electrical parameters.

## Input Features
- Background Voltage
- Pulse Voltage
- Pulse Frequency
- Pulse Duty Factor
- Wire Feed Rate
- Table Feed Rate
- RMS Current
- RMS Voltage

## Output
- UTS (MPa)

## Tools Used
- Python
- NumPy
- Pandas
- scikit-learn
